# Bill of Materials (Master List)

This list contains all the hardware, milled parts, and printed components required to assemble the A3DP Gantry. These quantities are cumulative for the entire project.

*Note: You will find smaller, sub-assembly-specific BOMs within the [Assembly Guide](Assembly_Guide.md) for easier step-by-step building.*
