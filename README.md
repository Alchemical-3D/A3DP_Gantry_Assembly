## 🏗️ About the A3DP Gantry

The A3DP Gantry provides a step-by-step framework to assemble a rigid, precision-driven CoreXY system. The design is broken down into modular sub-assemblies, providing an organized approach to building the machine. 

*The machine you are building now will be a reflection of the effort committed to assembling it. Attention to detail is directly related to both the performance of the machine and its longevity.*

## ✨ Key Technical Specifications

- **Drive System:** Standard CoreXY routing (A/B belt paths).
- **Toolhead Compatibility:** Ready to accept Alchemical-3D toolheads (e.g., Alembic Toolhead) and A3DP standards.
- **Hardware Profile:** Utilizes M3 and M5 hardware with precise milled aluminum constraint plates and standard MGN12H linear rails.
- **Tensioning System:** Front-mounted, incrementally adjustable belt tensioners.
